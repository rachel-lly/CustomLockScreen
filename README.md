# 倒计时

倒计时是一个采用Kotlin、MVVM架构开发的轻量高效的代办清单、日程管理和事项提醒工具。

## 主要功能

- 事件管理
- 分类本管理
- 事件提醒
- 事件分享
- 备份数据管理
- 日/夜间模式
